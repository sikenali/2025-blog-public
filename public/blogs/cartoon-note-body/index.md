## 重蹈覆辙

曾经学习建模，或者是建模玩的时候，总是会纠结一个问题：“*到底要调到什么程度是对的、好的？够的。*”

就像是以前的另一个问题，“*怎么样的设计，是美的？*”

如果说，差不多就行，但是，美这个事情，是需要大部分人同意的。

## 定义

我做出来，觉得好看的东西。我觉得它美，然后获得了多数人赞扬，那我就认可了自己的美是合适的、对的。

在 public 之前，我现在调制的模型，是对的么。或许这是每个人在做的时候都会遇到的问题，似乎没有答案。

算了，不再纠结。絮絮叨叨结束。

## 模型

我定下的目标是，做卡通人物建模，这样可以在 web 中达到**低模可爱**，受欢迎的效果。

做人物建模，难免时不时会结合现实人物特点，比如肩平还是有起伏，我定义：

“**减少构造为准，圆圆，平滑为准**”，先最小细节，再在运行中，调整可以优化的东西。

**不以视频教程为准，以我为准**

### 俯视图

虽然画好正视图和俯视图结构，调整后会有方方正正的感觉。因为俯视角下的圆滑没有调整。

这个时候进入雕刻的操作下的话，就会又陷入调整的漩涡。所以采用公式化的圆弧调整。

1. x 轴视图，alt + z 透明化
2. 对每一层进行调整，所以旋转一层，然后反转（ctrl + i）
3. h 隐藏其它点，对单独一层进行圆弧调整。alt + h 进行显示观察。

![](/blogs/cartoon-note-body/94f5803ea8617a38.webp)

### 空间调整

上面三个视图调整依旧挺好了，但空间上可以再调整下，比如鼓鼓的肚子，翘翘的屁股，瘦腰之类的。

依旧需要限制 单轴或平面 xy 平面上移动，这样改动点会更规律、可控一些。

## 技巧

* 身体延申出的手脚，使用**圆环**效果，然手脚更圆鼓鼓。
* **g** 之后按 x 就会限制在 x 轴移动，同时 **shift + x** 就会限制在 yz 平面限制移动
* **法线显示**，需要进入物体的编辑模式，右上角会出现“*网格编辑模式叠加层*”，里面就有法向线。还有测量更多数据

![](/blogs/cartoon-note-body/8401f6046f1dfcdd.webp)

* 右上角的 *视图叠加层* 还有可以留意的数据控制，包括辅助线，选择轮廓，选择中心点等。这里的栅格是指在单轴视图下的网格。
